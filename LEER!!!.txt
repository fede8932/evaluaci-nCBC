ESTA ES LA VERSION DOCKERIZADA DE LA PRUEBA PARA PUESTO DESARROLLADOR DE LA EMPRESA CBC

ESTE REPO CUENTA CON DOS RAMAS. LA RAMA MAIN, CONTIENE LA APLICACIÓN PARA EJECUTAR DIRECTAMENTE EN NODE DESDE LA COMPUTADORA FÍSICA.
PARA EJECUTAR EL PROYECTO DE ESTA MANERA, ES NECESARIO TENER INSTALADO NODEJS Y POSTGRES. ADEMÁS ES NECESARIO CREAR LA BASE DE DATOS Y CONFIGURAR
EL ARCHIVO .ENV QUE SE ENCUENTRA EN EL DIRECTORIO RAÍZ

LA RAMA APPDOCKER ES LA QUE CONTIENE LA BASE DE DATOS Y LA DEMO EN CONTENEDORES DE DOCKER.
PARA EJECUTARLA ES NECESARIO TENES INSTALADO DOCKER Y DOCKER COMPOSE EN LA COMPUTADORA (ACLARO QUE ESTA APP 
ESTA PROBADA EN DOCKER VERSION 20.10.18 Y DOCKER COMPOSE VERSION 2.10.2 ----IMPORTANTE----- UTILICE DOCKER COMPOSE Y NO DOCKER-COMPOSE)


1) EJECUTAR SIN DOCKER

-INSTALAR NODE JS Y POSTGRES
-DESCARGAR LA RAMA MAIN DEL REPO : https://github.com/fede8932/evaluaci-nCBC.git
-CREAR UNA BASE DE DATOS EN POSTGRES
-CONFIGURAR EL ARCHIVO .ENV CON LAS CREDENCIALES DE LA BASE DE DATOS
-INGRESAR EN LA CARPETA DEL PROYECTO
-EN LA CONSOLA EJECUTAR EL COMANDO npm install PARA INSTALAR LAS DEPENDENCIAS
-LUEGO EJECUTAR npm run build PARA COMPILAR EL PROYECTO
-EJECUTAR npm run start:prod PARA EJECUTAR EL PROYECTO COMPILARO
-SI DESEA CORRRLO EN MODO DESARROLLO EJECUTAR npm run start:dev
-PARA REALIZAR TEST DE LOS CONTROLADORES, EJECUTAR npm run test (EN ETAPA BETA)
-SE PUEDE ACCEDER A LA DOCUMENTACION DEL PROYECTO DESDE http://localhost:3001/api EL PUERTO 3001 VIENE CONFIGURADO POR DEFECTO, PUEDE MODIFICARLO DESDE 
EL ARCHIVO .ENV


2) EJECUTAR CON DOCKER COMPOSE

-INSTALAR DOCKER Y DOCKER COMPOSE (SE RECOMIENDAN LAS VERSIONES MENCIONADAS)
-DESCARGAR LA RAMA appDocker DEL REPO : https://github.com/fede8932/evaluaci-nCBC.git
-INICIAR LA CONSOLA EN LA RAIZ DEL PROYECTO
-EJECUTAR EL COMANDO docker compose o docker-compose DEPENDIENDO DE QUE HERRAMIENTA DE ORQUESTACIÓN DE CONTENEDORES TENGA INSTALADO.
-SE PUEDE ACCEDER A LA DOCUMENTACION DEL PROYECTO DESDE http://localhost:3001/api EL PUERTO 3001 VIENE CONFIGURADO POR DEFECTO, PUEDE MODIFICARLO DESDE 
EL ARCHIVO .ENV
